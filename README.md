# Sparrow Acccountant
- Sparrow Acccountant is pro tools for sparrow's wife

## Technologies
- Stack: MERN
- API: GraphQL
- UI: Ant Design Pro

## How to run on locally
- Server will run port 4000
- Client will run port 3000
### Before start
```
cd server
npm install
cd client
npm install
```
### Start
```
cd server
npm run dev
```

## How to deploy
- Run command
```bash
./deploy.sh
```
